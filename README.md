# GeophysicsDesktop
Проект направленности профиля, группа БПМИ-ПТ-23

Этот репозиторий содержит код для десктопного приложения, визуализирующее
геофизические поля в магнитных координатах.

## Цели продукта
1. Оптимизация процесса вычисления данных для построения графиуов и изображений;
2. Понятный, доступный и наглядный формат графиков и изображений;
3. Удобный интерфейс для пользователя.

**Данное приложение использует:**
- Язык программирования C++.
- СУБД Oracle Database для развертывания на win.
- Фреймворки: QT, VTK.
